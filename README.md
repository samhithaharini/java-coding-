##  What is Java?

Java is a **high-level, object-oriented, platform-independent programming language** developed by Sun Microsystems (now owned by Oracle).

It follows the principle:  
**"Write Once, Run Anywhere (WORA)"** ‚ÄĒ meaning compiled Java code can run on any platform that supports Java without recompilation.
